#  Ecommerce Cart 

This is a simple **Ecommerce Cart Application** built using **HTML, CSS, and JavaScript**.
It allows users to view products and add/remove items from the cart.

---

# Features

*  Display list of products
*  Add products to cart
*  Remove products from cart
*  Calculate total price
*  Simple and clean UI

---

# Technologies Used

* HTML
* CSS
* JavaScript (DOM Manipulation)

---

